# Exemplos de Planos de Site - FVS Consórcios

Este diretório contém 3 exemplos de sites baseados nos planos oferecidos aos clientes.

## 1. Mini Site (Básico)
**Pasta:** `/site` (site principal)
- Design simples e direto.
- Foco em conversão rápida (WhatsApp).
- Carregamento instantâneo.
- Ideal para cartões de visita digitais.

## 2. Landing Page Profissional
**Pasta:** `/Profissional`
- Design estruturado com navegação.
- Seções bem definidas (Sobre, Serviços, Parceiros).
- Grid de serviços com ícones.
- Mais credibilidade e informações visuais.

## 3. Landing Page Premium
**Pasta:** `/Premium`
- Design de alto padrão com animações.
- Vídeo de fundo na seção principal.
- Elementos interativos (Acordeão de FAQ, Contadores).
- Menu transparente que muda ao rolar.
- Experiência de usuário (UX) superior.

---
**Instruções:**
Basta abrir o arquivo `index.html` de cada pasta no navegador para visualizar o resultado.
